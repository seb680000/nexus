# Nexus Concept

Application Nexus publiee avec GitHub Pages.

## Mise en ligne

1. Creer un nouveau depot GitHub, par exemple `nexus-concept`.
2. Envoyer le contenu de ce dossier dans le depot.
3. Dans GitHub, ouvrir `Settings` > `Pages`.
4. Dans `Build and deployment`, choisir `GitHub Actions`.
5. Le workflow `Deploy Nexus to GitHub Pages` publie automatiquement `index.html`.

## Fichiers

- `index.html` : application Nexus.
- `.github/workflows/pages.yml` : deploiement GitHub Pages.

## Mise a jour

Remplacer `index.html` par la nouvelle version de `NEXUSCONCEPTBETA.html`, puis pousser les changements sur GitHub.
